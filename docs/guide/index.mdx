# 发布 npm 包

## 发布 npm 的包的好处是什么

- 方便团队或者跨团队**共享代码**，使用 npm 包就可以方便的管理，并且还可以进行**版本控制**
- 做**开源造轮子必备技术**，否则你做完的**轮子**如何让别人使用，难道是 U 盘拷贝？
- 面试很经常问到
- 增加**个人 IP**让更多的人知道你的技术能力和贡献

## 发布前准备工作

```bash
npm adduser
```

首先先检查一下是否是 npm 源然后创建一个 npm 账号
